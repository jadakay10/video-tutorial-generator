# Video Tutorial Generator

Automatically convert any `.mp4` screen recording or tutorial video into a detailed, step-by-step written tutorial in Markdown format. The script extracts audio, transcribes it with OpenAI Whisper, captures frames, and sends everything to Claude to generate a structured guide.

## What It Does

1. Extracts audio from your `.mp4` using ffmpeg
2. Captures one screenshot every 5 seconds using ffmpeg
3. Transcribes the audio locally using OpenAI Whisper (`base` model)
4. Sends the frames + transcript to Claude (`claude-sonnet-4-5`) via the Anthropic API
5. Streams a detailed Markdown tutorial to your terminal and saves it as `<video-name>.md`

## Requirements

- Python 3.10+
- [ffmpeg](https://www.gyan.dev/ffmpeg/builds/) installed and on your PATH
- An [Anthropic API key](https://console.anthropic.com/)

## Setup

### 1. Clone the repository

```bash
git clone https://github.com/jadakay10/video-tutorial-generator.git
cd video-tutorial-generator
```

### 2. Create and activate a virtual environment

**Windows:**
```bash
python -m venv venv
venv\Scripts\activate
```

**macOS/Linux:**
```bash
python -m venv venv
source venv/bin/activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Create your `.env` file

Create a file named `.env` in the project root and add your Anthropic API key:

```
ANTHROPIC_API_KEY=your-api-key-here
```

> Get your API key from [console.anthropic.com](https://console.anthropic.com/)

## Usage

```bash
python video_to_tutorial.py path/to/your/video.mp4
```

**Example:**

```bash
python video_to_tutorial.py demo.mp4
```

The script will print progress updates as it works:

```
[1/5] Extracting audio from 'demo.mp4'...
[2/5] Extracting one frame every 5 seconds...
[3/5] Transcribing audio with Whisper (base model)...
[4/5] Encoding 24 frames for the API...
[5/5] Sending to Claude to generate tutorial (streaming)...
```

When complete, a Markdown file named after your video (e.g. `demo.md`) will be saved in the project folder containing both the transcript and the generated tutorial.

## Output

The output `.md` file contains:

- **Transcript** — the full audio transcription from Whisper
- **Tutorial** — a structured, step-by-step guide generated by Claude, referencing the video frames

## Notes

- Whisper's `base` model runs locally — no internet required for transcription
- Longer videos will produce more frames and a larger API request
- The `frames/` folder is created automatically and holds the extracted JPEGs
- Your `.env` file is excluded from version control via `.gitignore`
